

# 🔗 Identity Reconciliation API

## 📌 Description

The **Identity Reconciliation API** is a backend service that links multiple contact records belonging to the same user using **email** and **phone number**.

When different requests contain overlapping contact information, the system intelligently determines the **primary contact** and connects additional records as **secondary contacts**, returning a **consolidated contact profile**.

This helps maintain a **unified user identity** even when customers interact with the system using different contact details.

## 📜 Features

* 🔍 Identify users using **email or phone number**
* 🧠 Automatically determine **primary and secondary contacts**
* 🔗 Merge multiple records belonging to the same user
* 📊 Return consolidated contact information
